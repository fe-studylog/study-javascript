JS can change HTML content

Hi I'm Spongebob!

<p id="p1">Hi I'm Spongebob!</p>
<button type="button" onclick="document.getElementById('p1').innerHTML = 'I love You~'">click me!</button>

JS can change HTML attribute values

<p><img id="myImg" src="https://picsum.photos/id/237/150/150'"></p>
<button type="button" onclick="document.getElementById('myImg').src = 'https://picsum.photos/seed/picsum/250/150'">click me!</button>

JS can change HTML styles (CSS)

font size 25px

<p id="p2">font size 25px</p>
<button type="button" onclick="document.getElementById('p2').style.fontSize ='25px'">click me</button>

JS can hide HTML elements

I'll disappear yap!

<p id="p3">I'll disappear yap!</p>
<button type="button" onclick="document.getElementById('p3').style.display ='none'">click me</button>

JS can show HTML elements

I'm here!

<p id="p4" style="display:none;>I'm here!</p>
<button type="button" onclick="document.getElementById('p4').style.display ='block'">click me</button>

JS in <body>
